# Hudhud

Hudhud is a desktop prayer-times app with an adorable hudhud (hoopoe) animated companion. The hudhud's main job is to make sure you keep up with your prayers.

How proud it looks when you complete a salaa on time:

![alt text](screenshot.png)

It is completely free and you can download it from the [releases](https://github.com/ziadh/Hudhud/releases) page.

If you want a certain feature added, please suggest it in the [issues](https://github.com/ziadh/Hudhud/issues) page
